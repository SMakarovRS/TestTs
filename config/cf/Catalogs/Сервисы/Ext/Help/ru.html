Справочник "Сервисы"

Справочник "Сервисы" предназначен для хранения каталога сервисов SLA.

При этом различают "Пользовательский" и "Внутренний" сервисы. Пользовательские видны конечному пользователю и он может выбрать из них. Это то, что написано на языке рядового пользователя. Например к таким относится сервис "Настройка компьютера". А внутренние сервисы предназначены только для сотрудников, которые оказывают услуги. Например "Настройка DNS".

Статус сервиса так же влияет на его видимость. Если сервис проектируется или не используется, то он не будет доступен для выбора в других документах.

В справочнике отражается информация:

  1. О тарифах сервиса (цены для разных категорий клиентов).
  2. О компетенциях (знания, которыми нужно обладать специалистам, чтобы оказывать услуги по этому сервису).
  3. Зависимости (от каких других сервисов зависит данный сервис).
  4. Процессы (в каких процессах может встречаться сервис).
