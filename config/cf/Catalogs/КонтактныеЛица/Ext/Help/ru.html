Справочник "Контактные лица"

Справочник предназначен для хранения физических лиц, с помощью которых осуществляется связь между контрагентом и предприятием. Данный справочник подчинен справочнику "Контрагенты".

Переход к списку договоров с контрагентом производится из панели навигации / Группа "Перейти"/ "Контактные лица" в форме элемента соответствующего справочника. 

 

Список контактных лиц контрагента определяется теми контактными лицами, которые подчинены соответствующему элементу справочника "Контрагенты".

Редактирование информации о контактных лицах контрагента производится в отдельном диалоговом окне, открываемом из формы списка справочника контактные лица.

При этом указывается информация, характеризующая контактное лицо:

  • Представление. Фамилия, имя и отчество контактного лица.
  • Физ. лицо. Полная информация о контактном лице. Выбирается из справочника "Физические лица".
  • Ответственный. Информация о сотруднике ответственном за ввод текущего документа.
  • Дата регистрации связи. Дата регистрации связи либо ее появление.
  • Дата прекращения связи. Дата регистрации прекращения связи или регистрация ее прекращения.

На закладке "Роли контактного лица" указываются роли контактного лица по классификации компании.

На закладке "Комментарий" указываются комментарий сотрудника, зарегистрировавшего связь.

На закладке "Адреса, телефоны" указывается контактная информация контрагента: E-mail, телефон.

На закладке "Дополнительные реквизиты" указываются дополнительные реквизиты, определяемые пользователем.
