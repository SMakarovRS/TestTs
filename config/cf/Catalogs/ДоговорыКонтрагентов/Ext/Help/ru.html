Справочник "Договоры контрагентов"

Справочник предназначен для хранения договоров расчетов, заключенных с контрагентами. Данный справочник подчинен справочнику "Контрагенты".

Переход к списку договоров с контрагентом производится из панели навигации / Группа "Перейти"/ "Договоры контрагентов" в форме элемента соответствующего справочника. 

 

Список договоров с контрагентом определяется теми договорами, которые подчинены соответствующему элементу справочника "Контрагенты".

Для каждого контрагента можно выбрать т.н. договор по умолчанию, который будет подставляться по умолчанию в документы.

Аналитический учет расчетов с контрагентами ведется по каждому договору.

 

Редактирование информации о договорах с контрагентом производится в отдельном диалоговом окне, открываемом из формы списка справочника договоров с контрагентами.

При этом указывается информация, характеризующая договор:

  • № договора. Указывается номер договора.
  • От. Указывается дата заключения договора.
  • Валюта расчетов по договору с контрагентом. Выбирается из справочника "Валюты".
  • Расчеты в у.е. Признак расчетов по договору в условных единицах.
  • Реализация на экспорт. Устанавливается, если по договору с покупателем осуществляются операции по ставке НДС 0%.
  • Организация. Указывается организация, с которой заключен договор. Выбирается из справочника "Организации".
  • Срок оплаты от покупателя. Указывается срок отсрочки платежа по договору с покупателем.
  • Срок оплаты поставщику. Указывается срок отсрочки платежа по договору с поставщиком.
  • Вид цен, Вид скидки, Вид цен контрагента. Значения по умолчанию. Подставляются в документы при выборе договора.
  • Вид договора с контрагентом: с поставщиком, с покупателем, с комитентом, с комиссионером или прочее. Реквизит не является обязательным к заполнению и используется только при обмене данными с конфигурацией "Бухгалтерия предприятия".
