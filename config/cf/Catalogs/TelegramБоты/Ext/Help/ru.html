Справочник "Боты Telegram"

Предназначен для ведения и инструкции по созданию ботов Telegram.

Реквизиты:

* Активность - включить или выключить бота.

* Ограниченный доступ - доступ к боту будет ограничен. Новых пользователей необходимо будет администратору "активировать" в справочнике "Пользователи Telegram"

* Способ взаимодействия с ботом - Как бот будет взаимодействовать с конфигурацией (периодический запрос новых событий или webhook).

* Токен - строковой реквизит, который назначается боту при создании бота и отвечает за связь с ботом Telegram.

Для способа взаимодействия с ботом через webhook:

* Адрес - HTTPS-адрес, на который бот будет отправлять обновления, без секретного ключа.

* Ключ - секретный ключ, по которому будут идентифицироваться сообщения Telegram к этому боту. Ключ должен быть уникальным среди всех Ваших использующих вебхук ботов.

* Использовать самоподписанный сертификат - сертификат создан самостоятельно.

* Сертификат - сам сертификат.
