Справочник "Этапы процессов"

Справочник "Этапы процессов" предназначен для хранения определенных этапов процесса - точек, которые ведут процесс от начала до его завершения.

  • Состояние этапа - предопределенный список состояний
  • Картинка - изображение по текущему этапу
  • Комментарий - комментарий этапа
  • Цвет фона - цвет фона для этапа
  • Цвет текста - цвет текста для этапа
  • Вид этапа - вид этапа процесса
