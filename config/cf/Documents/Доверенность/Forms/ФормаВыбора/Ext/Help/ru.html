Предназначен для выбора документа Доверенность.

Открывается по кнопке  после выбора типа данных Доверернность.

В списке выводится:

  • Дата - дата создания документа;
  • Номер - присваивается автоматически;
  • Организация - в поле указывается организация;
  • Доверенное лицо - на кого выписывается доверенность;
  • Банковский счет - банковский счет организации;
  • Контрагент - контрагент для которого выписана доверенность;
  • Договор - договор с контрагентом;
