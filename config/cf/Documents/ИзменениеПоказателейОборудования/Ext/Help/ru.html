Документ "Изменение показателей оборудования"

Документ предназначен для установки значений показателей оборудования. С помощью данного документа можно вести учет числовых показателей оборудования, таких как: количество заправок картриджей, количество отпечатанных листов и т.д. Т.е. это какой то числовой показатель, который будет накапливается со временем. Для просмотра сколько всего раз был заправлен картридж, отпечатано листов и т.д. воспользуйтесь отчетом Показатели оборудования. При этом показатели могут быть увеличены с учетом текущих значений (приплюсованы к показателям, которые были введены ранее), так и установлены по новой.

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Показатели реквизиты табличной части:

    • Дата изменения - когда изменился показатель. 
    • Номенклатура - собственно номенклатура показателя.
    • Карточка номенклатуры -
    • Вид показателя - что изменяем
    • Вид значения - что изменяем
    • Прошлое значение - его значение указанное в прошлый раз
    • Аналитика показателя - произвольная аналитика
    • Инициатор изменения - кто изменил показатель
    • Значение - собственное новое значение показателя или дельта для увеличения показателя
    • Описание - Комментарий к строке

Документ может быть введен на основании документов:

  • Задание
  • Изменение показателей оборудования
  • Поступление
  • Окончание обслуживания
