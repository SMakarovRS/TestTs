Документ "Закрепление сотрудников"

Документ предназначен для закрепления сотрудников за местами хранения. Так как в конфигурации поддержваиется связь Номенклатура <-> Место хранения <-> Сотрудник, то указанный документ предназначен для создания связи с выделенным.

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Сотрдуники реквизиты табличной части:

    • Место хранения - к какому месту хранения будет закреплен сотрудник.
    • Сотрудник - кто будет ответственным за место хранения (МОЛ).
    • Описание - Комментарий к строке

Документ может быть введен на основании документов:
