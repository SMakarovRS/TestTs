«Объединить все строки верхнего уровня в комплекты» - означет, что после проведения документа в остатках будут отражены только верхние строки в дереве номенклатуры (комплекты).
Вся остальная номенклатура, которая содержится внутри комплектов будет сохранена, но не будет показываться в остатках. Данную номенклатуру, всегда можно будет достать из комплектов документом "Разбиение комплектации". А так же добавить в комплект документом "Сборка (комплектация)".

Пример. К нам пришел системный блок "X7000B / ULTIMATE (X7352PRi): Core i7-3970X / 16 Гб / 256 Гб SSD + 2 Тб / 3 Гб RADEON HD7970 / DVDRW".
Мы внесли его в документ "Поступление", в котором получилось несколько строк:

  1. Корпус ASUS
  2. X7000B
  3. ULTIMATE (X7352PRi): Core i7-3970X
  4. Kingston 16 Гб
  5. 256 Гб SSD
  6. HDD WD 2 Тб
  7. 3 Гб RADEON HD7970
  8. DVDRW

Далее мы можем поступить следующим образом:

  1. Вести учет по каждой номенклатуре, т.е. "по частям" и в остатках будет вся номенклатура, которая указана в документе (галочка "Объединить все строки верхнего уровня в комплекты" не установлена).
  2. Вести учет только по поступившему системному блоку (при этом данные по тому, что внутри системного блока, то же будут сохранены, но отображаться в остатках не будут, их в любой момент можно "достать" из системного блока).
    Для этого необходимо в документ добавить еще одну строку: "X7000B / ULTIMATE (X7352PRi): Core i7-3970X / 16 Гб / 256 Гб SSD + 2 Тб / 3 Гб RADEON HD7970 / DVDRW" в нее перетащить всю номенклатуру (сделать ее родительской для всей остальной номенклатуры) и установить галочку "Объединить все строки верхнего уровня в комплекты", после чего провести документ. После этого мы увидим в остатке только один комплект, который мы добавили последним. Вся остальная номенклатура в остатках показываться не будет, но она в любой момент может быть извлечена из комплекта.
