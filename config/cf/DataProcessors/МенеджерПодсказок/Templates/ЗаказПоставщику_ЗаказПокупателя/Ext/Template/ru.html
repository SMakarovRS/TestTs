«Заказ покупателя» - источник потребности, под который оформляется заказ поставщику.

В реквизите Заказ покупателя можно указать заказ, для выполнения которого будут использоваться заказанные у поставщика товары, т. е. разместить заказ покупателя в заказе поставщику. Тогда при оформлении документа «Поступление запасов» по заказу поставщику, в котором размещен заказ покупателя, номенклатурные позиции будут автоматически резервироваться под данный заказ клиента на складе.

Для анализа размещения запасов существует отчет Ведомость по размещению запасов (выбирается из панели «Отчеты по снабжению и закупкам» в разделе «Закупки»).
