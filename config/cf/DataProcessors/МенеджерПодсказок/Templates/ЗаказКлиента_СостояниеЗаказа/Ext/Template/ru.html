«Состояние заказа» - это набор параметров, определяющих статус, в котором каждый момент времени находится заказ, например, выполнен ли заказ или только ожидает назначения, определяется его состоянием. Пользователь может создать любые удобные для работы значения состояний.

Каждому состоянию присваивается статус:

  • Открыт – заказ будет играть роль предварительной заявки.
  • В работе – данные заказа будут отражаться в учете.
  • Выполнен – устанавливается пользователем при выполнении заказа. Для закрытия заказа, имеющего состояние «Выполнен», следует установить флажок «Закрыт».

Для отмены заказа, имеющего состояние «Открыт» или «В работе», следует установить флажок «Закрыт».

Для списка заказов доступен быстрый отбор и назначение цвета представлению документа по состояниям.

Состав списка состояний определяется содержимым справочника «Состояния заказов клиентов».

Состояние по умолчанию для нового заказа клиента определяется в настройках пользователя.
