Обработка позволяет рассчитать количество релизов при обновление типовых.
Так же с помощью обработки можно обновить список существующих релизов, и возможные переходы между ними, со страницы на сайте ИТС указанной в карточке конфигурации.

По умолчанию логин и пароль ИТС берется из настроек текущей базы.

Текущая версия указывается версия с которой начинается обновление.
Требуемая версия - версия на которую нужно обновиться. (По умолчанию последний релиз)

Если в описании версии есть дополнительная информация, она выводится в виде текста.

При расчете обновлений отображается весь список промежуточных релизов.
